# NLM
A server-authoritative version control system built around explicit checkouts, differential storage, and thin clients. \

## Features
- Server-authoritative repository model  
- Checkout sessions with optional locking and sparse scopes  
- Differential, compressed server-side storage  
- CLI for scripting and power users  
- GUI for visual diffing, history browsing, and workspace management
